Assignment: Dictionaries and List Slicing in Python

Task 1: Student Marks Dictionary
Create a dictionary with student names as keys and marks as values.
Ask the user for a student’s name.
Display the marks if the student exists.
Show a “not found” message if the name is missing.

Task 2: List Slicing
Create a list of numbers from 1 to 10.
Extract the first five elements.
Reverse the extracted elements.
Print both the original slice and the reversed slice.
